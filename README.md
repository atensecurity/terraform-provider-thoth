# terraform-provider-thoth

Official Terraform provider source for Thoth headless control-plane operations.

## Scope

This repository will host:

- Provider schema and resources
- Versioned Terraform Registry publishing workflow
- Examples for tenant bootstrap, MDM sync, and SIEM webhook configuration

## Initial resources target

- `thoth_tenant_settings`
- `thoth_mdm_provider`
- `thoth_mdm_sync`
- `thoth_webhook_test`
